 ---
 Data Flow

 POST /api/analyses  (multipart: file=data.csv)
   → StoreAnalysisRequest validates (422 on failure)
   → AnalysisController::store()
       → TaskService::createAnalysisTask($user, $file)
           1. Task::create — type=data_analysis, status=pending, progress=0
           2. $file->store('uploads/{task_uuid}/filename.csv', 'local')
           3. $task->update(['payload' => ['file' => $storedPath]])
           4. AnalyseDataJob::dispatch($task)
           5. try/catch: on failure → deleteDirectory + $task->delete() + rethrow
           6. return $task
   → 201 JSON (task record, status=pending)

 --- queue worker picks up job ---

 AnalyseDataJob::handle(DataAnalysisService $service)
   1. Guard: Task::find($task->id) — return if deleted
   2. Guard: status !== Pending — return if already advanced
   3. update [status=Processing, progress=0]
   4. $storedPath = $task->payload['file']
   5. $stats = $service->analyse(Storage::disk('local')->path($storedPath))
   6. update [progress=50]  ← coarse milestone during analysis
   7. $resultPath = 'analyses/{task_uuid}/result.json'
   8. Storage::disk('local')->put($resultPath, json_encode($stats, JSON_PRETTY_PRINT))
   9. update [status=Completed, progress=100, result_path=$resultPath]

 AnalyseDataJob::failed(Throwable $e)
   - Guard: Task::find — skip if deleted
   - update [status=Failed, error_message=$e->getMessage()]
